VASPLAB

version 1.1.0.0

by Max Radin

Functions for integrating data from the Vienna Ab initio Simulation Package (VASP) into MATLAB.

VASPLAB provides functions that allow data from the Vienna Ab initio Simulation Package (VASP) to be integrated into MATLAB.- Import CHGCAR, DOSCAR, EIGENVAL, LOCPOT, OSZICAR, OUTCAR, POSCAR/CONTCAR

vasp_xml( filename, param ) - VASP_XML Import data from a vasprun.xml file.
import_chgcar( filename ) - IMPORT_CHGCAR Import a VASP CHGCAR file.
import_eigenval( filename ) - IMPORT_EIGENVAL Import a VASP EIGENVAL file.
import_locpot( filename ) - IMPORT_LOCPOT Import a VASP LOCPOT file.
import_doscar( filename ) - IMPORT_DOSCAR Import a VASP DOSCAR file.
import_poscar( filename ) - IMPORT POSCAR Import a VASP POSCAR/CONTCAR file.
import_outcar( filename, param ) - IMPORT_OUTCAR Import data from a VASP OUTCAR file.
import_oszicar( filename ) - IMPORT_OSZICAR Import energies from a VASP OSZICAR file.
export_poscar( filename, geometry ) - EXPORT_POSCAR Export a geometry struct as a VASP POSCAR file.
